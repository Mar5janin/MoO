# Galactic Settlers

Turowa gra strategiczna 4X wzorowana na klasykach gatunku takich jak Master of Orion czy Stellaris.


### Instrukcja
Wybierz system z kolonią, wybierz planetę przyporządkuj populację, ustal kolejke budowy. Wybierz badania, wyślij flote na zwiady i kliknij zakończ turę. I powtarzasz.

### Kluczowe Mechaniki
- **Fog of War**
- **Walki**
- **Instalacje Kosmiczne**
- **Przeciwnik AI**
